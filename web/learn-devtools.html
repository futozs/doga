Visual Studio Code – a fejlesztői szerkesztő
Mi a VS Code?

A Visual Studio Code (röviden VS Code) a világ legnépszerűbb kódszerkesztője — ingyenes, gyors, és rengeteg kiegészítővel bővíthető. Webfejlesztéshez tökéletes választás: kiemelés, automatikus kiegészítés, és élő előnézet is elérhető benne.

Ha még nincs telepítve: code.visualstudio.com — Windows, Mac és Linux alá egyaránt ingyenesen letölthető.

Hasznos billentyűkombinációk
Többsoros szerkesztés

Az egyik leghasznosabb funkció: egyszerre több helyen írhatsz a fájlban.

Alt + Kattintás
Több kurzor egyszerre Kattints több sorra tartva az Alt-ot — mindenhol egyszerre írhatsz.
Ctrl + D
Következő egyező kijelölése Jelölj ki egy szót, majd Ctrl+D-vel a következő előfordulás is kijelölődik — így egyszerre átírhatod mindet.
Ctrl + Shift + L
Összes egyező kijelölése Egy lépésben kijelöli a fájlban szereplő összes egyező szöveget.
Ctrl + Alt + Le
Kurzor hozzáadása lejjebb Egymás alatti sorokhoz ad kurzort — hasznos ha több egymás alatti sort kell egyformán módosítani.
Keresés és csere
Ctrl + H
Keresés és csere Megnyílik a csere panel — pl. minden "szin" szót kicserélhetsz "color"-ra egyszerre.
Ctrl + F
Keresés a fájlban Megkeresi a beírt szöveget az aktuális fájlban. Enter-rel lép a következő találatra.
Ctrl + Shift + F
Keresés minden fájlban Az összes megnyitott fájlban keres — hasznos ha egy változónevet mindenhol át kell írni.
Ctrl + P
Gyors fájlnyitás Fájlnév begépelésével azonnal megnyit bármely fájlt a projektből, mappák böngészése nélkül.
Sorok kezelése és egyéb trükkök
Alt + Fel/Le
Sor mozgatása A kurzort tartalmazó sort felfelé vagy lefelé mozgatja anélkül, hogy ki-be kellene másolni.
Alt + Shift + Le
Sor duplikálása Az aktuális sort azonnal lemásolja az alá — gyors ha hasonló sorokat kell egymás után írni.
Ctrl + /
Sor kikommentezése Kommentté alakítja a sort (és vissza). HTML-ben <!-- -->, JS-ben // kerül elé.
Shift + Alt + F
Kód formázása Prettier telepítése után az egész fájlt rendezett, egységes formátumba hozza.
Parancspaletta: Ha nem emlékszel egy billentyűkombinációra, nyomd meg a Ctrl + Shift + P billentyűket — megkeresi és végrehajtja a parancsokat, rájuk kattintva a billentyűkombinációt is megmutatja.
W3Schools – a webfejlesztő enciklopédiája
Mi a W3Schools?

A W3Schools a világ egyik leglátogatottabb webfejlesztési oktatóoldala. Ingyenesen elérhető, minden HTML, CSS és JavaScript témáról tartalmaz:

  • Referencia táblázatokat – összes tag, tulajdonság, metódus listázva
  • Interaktív példákat – "Try it Yourself" gombbal kipróbálható kódok
  • Oktató feladatokat – kvízek, gyakorlatok
Ha elfelejtesz valamit (pl. melyik CSS tulajdonsággal lehet keretezni?), a W3Schools az első hely ahol érdemes keresni.

MDN Web Docs – a szakmai referencia
Mi az MDN?

Az MDN Web Docs (Mozilla Developer Network) a legteljesebb és legmegbízhatóbb webfejlesztési dokumentáció. A W3Schools-nál részletesebb, és a böngészők által ténylegesen implementált szabványokat írja le.

  • Minden CSS tulajdonságnál megmutatja, melyik böngésző támogatja
  • Pontos szintaxis leírások, edge case-ek
  • Haladóbb fejlesztőknek különösen hasznos

Validálás – miért fontos?
Mi a validálás?

A validálás azt jelenti, hogy ellenőrzöd: a kódod megfelel-e a hivatalos HTML/CSS szabványoknak.

Miért fontos?

  • A hibás kód különbözőképpen jelenhet meg különböző böngészőkben
  • Záratlan tagek, elírt attribútumok láthatatlan hibákat okozhatnak
  • A validátor megmutatja pontosan melyik sorban van a hiba
  • Keresőoptimalizálás (SEO) szempontjából is számít a tiszta kód
Mielőtt beadod a feladatot, mindig futtasd le a validátoron! Egy záratlan </div> az egész oldal elrendezését tönkreteheti.
Hogyan validálsz?
1
Nyisd meg a W3C HTML Validátort: validator.w3.org
2
Válaszd a "Validate by Direct Input" fület, és illeszd be a kódodat
3
Kattints a "Check" gombra – a hibák sorban jelennek meg piros színnel, sorszámmal
4
Javítsd a hibákat, majd futtasd le újra – addig amíg "No errors or warnings" üzenetet nem kapsz

Böngésző – fejlesztői eszközök (DevTools)
Mi a DevTools?

Minden modern böngészőbe (Chrome, Firefox, Edge) be van építve egy fejlesztői eszközkészlet, amellyel valós időben vizsgálhatod és módosíthatod az oldalak kódját.

Megnyitás: F12 vagy Ctrl + Shift + I

A legfontosabb fülek
  • Elements (Elemek) – az oldal HTML-CSS struktúrája; valós időben szerkeszthető
  • Console (Konzol) – JavaScript hibák és console.log() kimenetek
  • Sources – a betöltött fájlok (HTML, CSS, JS) listája
  • Network (Hálózat) – betöltési idők, kérések figyelése
Tipp: Az Elements fülön kattints egy elemre jobb gombbal az oldalon → "Inspect" (Vizsgálat) – azonnal az adott elem CSS-énél ugrik a DevTools.
Reszponzivitás ellenőrzése

A DevTools-ban a Ctrl + Shift + M gombbal (vagy a kis mobilikon ikonra kattintva) különböző képernyőméreteken nézheted meg az oldaladat – telefon, tablet, laptop nézetben egyaránt.
